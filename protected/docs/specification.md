Требования
================================================================================

* CRUD-операции для постов;
* свойства поста:
	* название;
	* дата создания;
	* дата изменения;
	* текст;
	* теги;
* виджеты:
	* облако тегов;
* дополнительно:
	* поддержка ката;
	* поддержка Markdown в текстах постов;
* настраиваемые параметры:
	* пароль админа;
	* число постов на страницу;
	* максимальное число тегов в облаке;
